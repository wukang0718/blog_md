### My97设置日期控件日期格式

**需要什么格式的只需要修改dateFmt的值就可以了**

    <input type="text" id="d233" onclick="WdatePicker({dateFmt:'yyyy-MM-dd HH:mm:ss'})"/>
    
#####  yMdHmswW分别代表年月日时分秒星期周,你可以任意组合这些元素来自定义你个性化的日期格式
